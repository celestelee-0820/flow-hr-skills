# 术语与分工

## JD（岗位说明书）

- **侧重**：岗位目的、职责、任职条件（学历/年限/证书等硬性要求可写清）、汇报关系、工作地点等 **岗位与组织要求**。
- 与胜任力分工：**硬性门槛、职责清单** 优先落在 JD；**可观察行为与能力维度** 落在胜任力模型。

## 胜任力模型

- **侧重**：维度（如「客户导向」「结构化问题解决」）、行为指标或分级描述、与岗位关键挑战的对应关系。
- 应与 JD 中的核心职责 **对齐**：胜任力覆盖「做好这份工作要展现的行为」，而非重复罗列 JD 里的硬性条件条文。

## 面试评价（面评）

- **侧重**：基于面试 transcript / notes 的 **行为证据**、定性推荐、分维度观察；不用综合分制作为录用结论（详见 [`../interview-evaluation/SKILL.md`](../interview-evaluation/SKILL.md)）。
- **行业怎么写、长度与禁忌**：以 [`industry-norms.md`](industry-norms.md) 为准；**评什么维度**：以 [`../interview-evaluation/dimensions.md`](../interview-evaluation/dimensions.md) 为准。
