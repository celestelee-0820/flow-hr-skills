# 面试评价输出模板

> 说明：若用户提供了自定义模板，优先使用用户结构；禁忌与长度仍建议遵守 [industry-norms.md](industry-norms.md)。  
> `[占位符]` 为待填项。  
> **全文不使用综合分制（如 1–5 分、X/5）**；录用相关结论用 **定性推荐** 与 **一句话优劣势**。

---

## 候选人模板路由

| 用户指定的沟通倾向 | 本节模板 |
|--------------------|----------|
| 推进 | 候选人·推进 |
| 待定 | 候选人·待定 |
| 不推进 | 候选人·不推进 |
| 用户未指定 | **候选人·待定**（中性兜底） |

候选人正文中 **不出现**「推荐录用/不推荐」等内部定性标签以外的过度承诺；不写分数或等级制综合结论。

---

## 候选人·推进

> 风格：肯定语气，亮点先行，可交代下一步；不提内部顾虑细节

```
[候选人姓名] 你好，

感谢你参加 [公司名/团队名] 关于 [岗位名称] 的面试。

在本次面试中，你给我们留下了深刻印象。[核心亮点1]，[核心亮点2]，这些能力与我们团队当前的需求高度契合。

[可选：具体行为亮点，1–2 句，有行为证据时添加]
例：在讨论 [具体场景] 时，你分享的 [关键做法/决策] 体现了你在 [能力维度] 上的积累。

我们很高兴通知你，你已进入下一阶段流程。[下一步说明，如：HR 将在 X 个工作日内与你联系，沟通后续安排。]

期待与你共事！
[面试官姓名 / 团队]
```

---

## 候选人·待定

> 风格：中性诚实，说明评估仍在进行；不传递过度期望或否定

```
[候选人姓名] 你好，

感谢你参加 [公司名/团队名] 关于 [岗位名称] 的面试。

你在面试中展现了[正面评价，如：扎实的基础和认真的思考态度]。我们目前正在综合评估与岗位匹配度，[说明待定原因，如：仍在完成多轮面试 / 岗位需求仍在确认 / 需补充某方面信息]，因此暂时无法给出最终结论。

我们预计将在 [时间范围，如：X 个工作日内] 与你同步进展，感谢你的耐心等待。

[面试官姓名 / 团队]
```

---

## 候选人·不推进

> 风格：建设性，聚焦岗位匹配与成长方向；不人身否定，不透露其他候选人信息

```
[候选人姓名] 你好，

感谢你花时间参加 [公司名/团队名] 关于 [岗位名称] 的面试，以及在面试中分享的思考。

经过综合评估，此次我们暂时无法向你推进后续流程。[简要说明方向，非定性，如：目前岗位对 [某能力/经验] 有较高要求，与本次面试中展现的重点尚有一定距离。]

从面试过程来看，你在 [优势点1] 和 [优势点2] 方面有可圈可点之处。针对后续发展，以下方向或许有参考价值：
- [成长建议1]
- [成长建议2，可选]

感谢你的投入，祝你在职业道路上顺利。如有机会，欢迎你关注我们日后的职位开放。

[面试官姓名 / 团队]
```

## 面试官存档

> 风格：中性客观、结构化；**定性推荐** + **一句话优势/风险** + 分维度观察；无数值综合分

```
## 面试评价记录

**基本信息**
- 候选人：[姓名]
- 应聘岗位：[职位名称] / [层级：Junior / Senior / Manager+]
- 行业语境：[互联网 / 消费品 / …]（可据 JD）
- 面试日期：[日期]
- 面试官：[姓名]
- 面试形式：[视频 / 现场 / 电话]

---

**定性推荐**（择一勾选或保留对应表述）

[ ] 推荐进入下一轮  
[ ] 待定（建议补充：[面试方向 / 材料 / 其他]）  
[ ] 不推荐

**一句话优势：**  
[客观单句，须有行为证据支撑]

**一句话风险或待验证点：**  
[客观单句；若无则写：暂无明显风险，或注明某维度证据不足建议复核]

---

**分维度观察**

| 维度 | 档位 | 关键观察 |
|------|------|----------|
| [维度1] | 显著优秀 / 良好 / 达标 / 需提升 / 明显不足 / 证据不足 | [1–2 句行为证据] |
| [维度2] | … | … |
| [维度N] | … | … |

> 档位说明：**显著优秀** 明显超预期；**良好** 达预期且证据充分；**达标** 基本满足；**需提升** 有差距且有依据；**明显不足** 关键缺口；**证据不足** 面试未展现、不推断

---

**核心优势（展开）**
1. [优势1：具体描述 + 证据]
2. [优势2：具体描述 + 证据]

**主要顾虑或待验证点（展开）**
1. [顾虑1：具体描述 + 证据，可选]
2. [顾虑2：可选]

---

**推荐依据摘要**  
[2–4 句：为何给出上述定性推荐，紧扣行为证据]
```

## HR 决策支持

> 风格：结论前置；**定性推荐** + **一句话优劣势** + 风险列表；无数值综合分

```
## 面试结论摘要（供 HR / 用人部门决策）

**候选人：** [姓名] | **岗位：** [职位] | **行业语境：** [简述] | **面试官：** [姓名] | **日期：** [日期]

---

### 定性推荐

**结论：** [推荐进入下一轮 / 待定 / 不推荐]

**一句话优势：**  
[单句，有证据]

**一句话风险或待验证点：**  
[单句；或：暂无明显风险 / 某维度证据不足建议补充面试]

---

### 关键优势（分条）

- [优势1]
- [优势2]

### 风险提示与建议关注点

- [风险或待验证项1；若无实质风险可写「暂无明显合规或录用风险，仍建议完成既定背调流程」]
- [风险或待验证项2，可选]

---

### 薪资与定级参考（可选）

[仅基于面试表现的客观描述，避免承诺具体数字；如：在 [维度] 上展现的经验深度与 JD 中 [要求] 的匹配度较高]

---

### 备注

[其他 HR 需知信息：其他 offer、时间窗口、候选人诉求等，勿写无依据判断]
```

---

## 跨部门同步

> 风格：简报；背景 + 面试印象 + **定性结论一句**；不写综合分数、少内部敏感细节

```
## 候选人面试情况同步

**候选人：** [姓名] | **岗位：** [职位] | **日期：** [日期]

**背景：**  
[1–2 句：来源与相关经历，可对照简历脱敏表述]

**面试表现：**  
[2–3 句：与接收方相关的观察，避免泄密]

**结论：**  
[一句话：如建议用人部门继续推进 / 建议暂缓 / 已与 HR 同步待定结论等]

**需跨部门配合（如有）：**
- [事项1]
```
