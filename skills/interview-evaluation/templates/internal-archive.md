# 模板 — 面试官存档（内部）

> 风格：中性客观、结构化；定性推荐 + 一句话优势/风险 + 分维度观察；无数值综合分。  
> 若有前序面试结论，在「基本信息」后增加「前序面试摘要」一节。

```
## 面试评价记录

**基本信息**
- 候选人：[姓名]
- 应聘岗位：[职位名称] / [层级：Junior / Senior / Manager+]
- 行业语境：[互联网 / 消费品 / …]
- 面试日期：[日期]
- 面试官：[姓名] / [面试轮次：一面 / 二面 / 三面]
- 面试形式：[视频 / 现场 / 电话]

---

[如有前序面试记录]
**前序面试摘要**
[简述既往轮次的核心结论与待验证点，1–3 句]

---

**定性推荐**（择一保留）

[ ] 推荐进入下一轮  
[ ] 待定（建议补充：[面试方向 / 材料 / 其他]）  
[ ] 不推荐

**一句话优势：**  
[客观单句，须有行为证据支撑]

**一句话风险或待验证点：**  
[客观单句；若无则写：暂无明显风险，或注明某维度证据不足建议复核]

---

**分维度观察**

| 维度 | 考察层 | 档位 | 关键观察（行为证据） |
|------|--------|------|---------------------|
| [维度1] | 深度/广度/高度 | 显著优秀/良好/达标/需提升/明显不足/证据不足 | [1–2 句 STAR 证据] |
| [维度2] | … | … | … |

---

**核心优势（展开）**
1. [优势1：具体描述 + 证据]
2. [优势2：具体描述 + 证据]

**主要顾虑或待验证点（展开）**
1. [顾虑1：具体描述 + 证据，可选]

---

**推荐依据摘要**  
[2–4 句：为何给出上述定性推荐，紧扣行为证据]
```
