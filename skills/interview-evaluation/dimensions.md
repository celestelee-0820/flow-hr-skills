# 评估维度框架

> 本文件定义各岗位层级的 **评估维度列表**（面评里「评什么」）。
> **各行业怎么写、写多长、哪些话不能说**：以 [industry-norms.md](industry-norms.md) 为准，避免与本文件重复或冲突。
> 若用户提供了自定义 rubric，以用户 rubric 为准，忽略本文件的维度表。

---

## 使用说明

- 每个维度对应一行记录：**维度 / 定性档位 / 行为证据**（不使用综合分制）
- 证据不足的维度：档位标注「证据不足」，不主观推断
- 文末 **行业附加维度** 仅补充「多评哪些维度」；该行业的 **语气、禁忌、长度** 仍以 industry-norms 为准

---

## Junior（应届 / 1-3 年经验）

> 核心考察：能否上手、能否成长、能否融入

| 维度 | 说明 | 关键观察点 |
|------|------|------------|
| **技术基础** | 与岗位直接相关的专业知识和技能 | 能否准确回答技术问题；基础概念是否扎实 |
| **学习能力** | 面对新知识/反馈的吸收和应用速度 | 追问后的反应；能否在面试中举一反三 |
| **问题解决** | 拆解问题和推进解决的思维方式 | 遇到不会的问题时的应对策略；逻辑清晰度 |
| **沟通表达** | 清晰、有条理地传达想法的能力 | 回答是否结构化；能否听懂问题意图 |
| **文化适配** | 与团队工作方式和价值观的匹配程度 | 对工作方式的期待；对反馈的接受态度 |
| **主动性** | 自发探索和推进的意愿 | 是否有自驱的项目/学习经历；对岗位的准备程度 |

## Senior（3-8 年经验，独立贡献者）

> 核心考察：能否独立交付、能否影响质量、能否跨协作

| 维度 | 说明 | 关键观察点 |
|------|------|------------|
| **技术深度** | 对核心专业领域的深度掌握和系统思考 | 能否讨论权衡取舍；有无对某领域的独到见解 |
| **交付质量** | 独立负责项目从启动到收尾的能力 | 有无端到端负责的项目；遇到阻碍如何处理 |
| **自驱力** | 无需大量监督下的主动推进能力 | 主动识别问题和机会的案例；如何设定优先级 |
| **跨职能协作** | 与不同角色（产品/设计/运营/其他技术）高效配合 | 有无跨职能项目经验；处理分歧的方式 |
| **影响力** | 通过专业能力影响团队方向和做法 | 有无推动技术/流程改进的案例 |
| **沟通与对齐** | 向上/向下/横向的沟通有效性 | 如何汇报进展；如何处理预期差异 |

## Manager+（团队负责人 / 部门 Leader）

> 核心考察：能否建立团队、能否做决策、能否承担业务结果

| 维度 | 说明 | 关键观察点 |
|------|------|------------|
| **领导力** | 激励、发展和保留团队成员的能力 | 团队规模和管理经验；如何处理低绩效成员 |
| **人才培养** | 识别潜力、帮助他人成长的意愿和方法 | 有无培养出晋升的成员；1:1 和反馈机制 |
| **商业判断** | 将业务目标转化为团队方向和优先级的能力 | 如何决定做什么/不做什么；有无数据驱动决策案例 |
| **战略思维** | 对所在领域的全局认知和前瞻判断 | 能否描述所在行业/赛道的趋势；有无前瞻性布局案例 |
| **跨组织协作** | 在多个团队/部门之间推动对齐的能力 | 如何处理资源竞争和优先级冲突 |
| **抗压与决策** | 在信息不完整或高压环境下的决策质量 | 最困难的一次决策案例；如何处理失败复盘 |
| **文化塑造** | 主动定义和传递团队文化和行为标准 | 团队文化的关键词；如何处理价值观分歧 |

## 行业附加维度（可选）

根据用户提供的行业，在基础维度之后附加以下维度：

### 互联网 / 科技
- **数据驱动**：是否习惯用数据支撑决策和评估效果（DAU、转化率、留存等）
- **产品意识**（非技术岗）：对用户需求和产品形态的理解深度

### 消费品 / 零售
- **市场洞察**：对消费者行为和渠道变化的敏感度
- **执行落地**：将策略转化为门店/动销/SKU 层级执行的能力

### 制造业
- **流程管控**：对生产/供应链/质量体系的熟悉程度
- **成本意识**：在设计和决策中主动考量成本效益

### 金融
- **合规意识**：对监管要求和合规操作的了解程度
- **风险判断**：识别和量化业务风险的能力

## 定性档位（统一，与 templates 中分维度表一致）

| 档位 | 含义 |
|------|------|
| 显著优秀 | 明显超出岗位预期，有具体行为证据支撑 |
| 良好 | 达到岗位预期，有较充分的行为证据 |
| 达标 | 基本满足要求，证据一般 |
| 需提升 | 有明显差距，有具体不足证据 |
| 明显不足 | 核心能力缺失，影响录用决策 |
| 证据不足 | 面试中未充分展现，无法判断 |
