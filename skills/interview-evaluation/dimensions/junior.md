# 维度表 — Junior（应届 / 1–3 年经验）

> 核心期待：能否上手、能否成长、能否融入

| 维度 | 说明 | 关键观察点 | 考察层 | 面试官类型 |
|------|------|-----------|--------|-----------|
| **技术基础** | 与岗位直接相关的专业知识和技能 | 基础概念是否扎实；追问后能否准确回答 | 深度 | 专业面 |
| **问题解决** | 拆解问题和推进解决的思维方式 | 遇到不会的问题时的应对策略；逻辑清晰度 | 深度 + 广度 | 专业面 |
| **沟通表达** | 清晰、有条理地传达想法的能力 | 回答是否结构化；能否听懂问题意图 | 广度 | 共同 |
| **学习能力** | 面对新知识 / 反馈的吸收和应用速度 | 追问后的反应；能否在面试中举一反三 | 高度 | 专业面 |
| **主动性** | 自发探索和推进的意愿 | 是否有自驱的项目/学习经历；对岗位的准备程度 | 高度 | 共同 |
| **文化适配** | 与团队工作方式和价值观的匹配程度 | 对工作方式的期待；对反馈的接受态度 | 高度 | HR 面 |
| **职业动机** | 选择此岗位/公司的原因及职业规划 | 动机是否清晰；期待与岗位是否匹配 | 广度 | HR 面 |
| **稳定性** | 预期在岗时长及转换意愿 | 离职原因；对短期/长期发展的预期 | 广度 | HR 面 |

## 定性档位

| 档位 | 含义 |
|------|------|
| 显著优秀 | 明显超出应届/低年资预期，有具体行为证据 |
| 良好 | 达到岗位预期，证据充分 |
| 达标 | 基本满足，证据一般 |
| 需提升 | 有明显差距，有依据 |
| 明显不足 | 核心能力缺失，影响录用 |
| 证据不足 | 面试中未充分展现，不推断 |
