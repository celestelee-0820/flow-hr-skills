# 维度表 — Senior（3–8 年经验，独立贡献者）

> 核心期待：能否独立交付、能否影响质量、能否跨协作

| 维度 | 说明 | 关键观察点 | 考察层 | 面试官类型 |
|------|------|-----------|--------|-----------|
| **技术深度** | 对核心专业领域的深度掌握和系统思考 | 能否讨论权衡取舍；有无独到见解 | 深度 | 专业面 |
| **交付质量** | 独立负责项目从启动到收尾的能力 | 有无端到端负责的项目；遇到阻碍如何处理 | 深度 | 专业面 |
| **跨职能协作** | 与产品 / 设计 / 运营 / 其他技术高效配合 | 有无跨职能项目；处理分歧的方式 | 广度 | 共同 |
| **沟通与对齐** | 向上 / 向下 / 横向的沟通有效性 | 如何汇报进展；如何处理预期差异 | 广度 | 共同 |
| **自驱力** | 无需大量监督下的主动推进能力 | 主动识别问题和机会的案例；如何设定优先级 | 广度 + 高度 | 专业面 |
| **影响力** | 通过专业能力影响团队方向和做法 | 有无推动技术 / 流程改进的案例 | 高度 | 专业面 |
| **职业动机** | 选择此岗位/公司的原因及职业规划 | 动机是否清晰；期待与岗位是否匹配 | 广度 | HR 面 |
| **稳定性** | 预期在岗时长及转换意愿 | 离职原因；对职业发展路径的预期 | 广度 | HR 面 |
| **文化适配** | 与团队工作方式和价值观的匹配程度 | 对协作风格的偏好；对反馈的接受态度 | 高度 | HR 面 |

## 定性档位

| 档位 | 含义 |
|------|------|
| 显著优秀 | 明显超出 Senior 预期，有具体行为证据 |
| 良好 | 达到岗位预期，证据充分 |
| 达标 | 基本满足，证据一般 |
| 需提升 | 有明显差距，有依据 |
| 明显不足 | 核心能力缺失，影响录用 |
| 证据不足 | 面试中未充分展现，不推断 |
