# 行业附加维度（按需叠加）

> 仅在已确认行业时读本文件；叠加到对应层级维度表之后。  
> 行业的语气、禁忌、长度规范见 `industry-norms/` 对应文件，不在本文件重复。

## 互联网 / 科技


| 维度             | 说明              | 关键观察点                   | 考察层 |
| -------------- | --------------- | ----------------------- | --- |
| **数据驱动**       | 用数据支撑决策和评估效果的习惯 | 能否举出具体指标案例（DAU、转化率、留存等） | 广度  |
| **产品意识**（非技术岗） | 对用户需求和产品形态的理解深度 | 能否描述用户视角；对产品迭代的主动参与     | 广度  |


## 消费品 / 零售


| 维度       | 说明                          | 关键观察点            | 考察层 |
| -------- | --------------------------- | ---------------- | --- |
| **市场洞察** | 对消费者行为和渠道变化的敏感度             | 能否举出实际观察到的市场变化案例 | 广度  |
| **执行落地** | 将策略转化为门店 / 动销 / SKU 层级执行的能力 | 有无一线落地案例；如何跟踪和复盘 | 深度  |


## 制造业


| 维度       | 说明                    | 关键观察点           | 考察层 |
| -------- | --------------------- | --------------- | --- |
| **流程管控** | 对生产 / 供应链 / 质量体系的熟悉程度 | 有无体系认证经验；如何处理异常 | 深度  |
| **成本意识** | 在设计和决策中主动考量成本效益       | 能否举出降本或工艺优化案例   | 深度  |


## 金融


| 维度       | 说明              | 关键观察点                | 考察层     |
| -------- | --------------- | -------------------- | ------- |
| **合规意识** | 对监管要求和合规操作的了解程度 | 主动提及合规节点的案例；对灰色地带的判断 | 深度 + 广度 |
| **风险判断** | 识别和量化业务风险的能力    | 有无风险识别和处置的具体案例       | 广度      |
