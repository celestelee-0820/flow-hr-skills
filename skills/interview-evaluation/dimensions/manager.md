# 维度表 — Manager+（团队负责人 / 部门 Leader）

> 核心期待：能否建立团队、能否做决策、能否承担业务结果  
> 注意：Manager+ 候选人即使是「一面」，面试官通常已是总监级，考察层从广度起步。  
> 考察层说明：**广度** = 业务负责人 / 总监重点；**高度** = 部门 GM / 高管重点

| 维度 | 说明 | 关键观察点 | 考察层 |
|------|------|-----------|--------|
| **领导力** | 激励、发展和保留团队成员的能力 | 团队规模和管理经验；如何处理低绩效成员 | 广度 |
| **人才培养** | 识别潜力、帮助他人成长的意愿和方法 | 有无培养出晋升的成员；1:1 和反馈机制 | 广度 |
| **商业判断** | 将业务目标转化为团队方向和优先级的能力 | 如何决定做什么 / 不做什么；有无数据驱动决策案例 | 广度 |
| **跨组织协作** | 在多个团队 / 部门之间推动对齐的能力 | 如何处理资源竞争和优先级冲突 | 广度 + 高度 |
| **抗压与决策** | 在信息不完整或高压环境下的决策质量 | 最困难的一次决策案例；如何处理失败复盘 | 高度 |
| **战略思维** | 对所在领域的全局认知和前瞻判断 | 能否描述行业 / 赛道趋势；有无前瞻性布局案例 | 高度 |
| **文化塑造** | 主动定义和传递团队文化和行为标准 | 团队文化的关键词；如何处理价值观分歧 | 高度 |

## 定性档位

| 档位 | 含义 |
|------|------|
| 显著优秀 | 明显超出 Manager+ 预期，有具体行为证据 |
| 良好 | 达到岗位预期，证据充分 |
| 达标 | 基本满足，证据一般 |
| 需提升 | 有明显差距，有依据 |
| 明显不足 | 核心能力缺失，影响录用 |
| 证据不足 | 面试中未充分展现，不推断 |
